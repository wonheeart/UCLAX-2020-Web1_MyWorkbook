
Essay: Read file relationships: (Site Relative, Document Relative, and Absolute Paths).

Site relative paths describes the path from a site's root folder all the way to the document.
This type of pathing is typically used in large websites that uses several servers.

Document relative paths are usually the most commonly used in today's websites.  This pathing
is particularly useful when current document and other linked documents are all under 
the same folder. When you use this type of pathing, you leave out the absolute URL and only 
provide the pathing that is different.

Absolute path refers to the type of pathing that needs complete details to locate a document
or a folder.  Most commonly, absolute paths are used to refer to links that are on a different
domain than your home domain.  It must contain the full path and always need to start from it's 
root directory.
